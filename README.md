## Features

### ( POST )

- 회원가입 
- 아이디 중복체크
- 이메일 중복체크
- 로그인
- 장바구니 담기
- 상품주문

### ( GET )

- 메인페이지
- 상세페이지
- 장바구니 내역
- 주문내역

### ( PATCH )

- 장바구니 수량변경
- 주문취소

### ( DELETE )

- 장바구니 삭제
